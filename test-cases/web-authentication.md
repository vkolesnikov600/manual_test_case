# Тест-кейсы: авторизация пользователя и безопасность

Набор покрывает базовую авторизацию, валидацию формы входа, управление сессией, защиту от brute force, SQL Injection, XSS, JWT и 2FA.

## TC-AUTH-001 - Успешная авторизация

**Приоритет:** High
**Тип:** Smoke, Functional
**Статус:** Not run

### Предусловия

- Пользователь зарегистрирован и активен

### Тестовые данные

- Валидный логин, валидный пароль

### Шаги

1. Открыть страницу авторизации
2. Ввести корректный логин
3. Ввести корректный пароль
4. Нажать кнопку `Войти`
5. Проверить страницу после входа

### ОР

1. Страница открыта, форма входа отображается
2. Значение отображается в поле логина
3. Пароль скрыт маской
4. Пользователь успешно авторизован
5. Выполнен переход на главную страницу или в личный кабинет, отображается имя пользователя
