# Тест-кейсы: авторизация пользователя и безопасность

Набор покрывает базовую авторизацию, валидацию формы входа, управление сессией, защиту от brute force, SQL Injection, XSS, JWT и 2FA.

## TC-AUTH-001 - Успешная авторизация

| Поле | Значение |
| --- | --- |
| Приоритет | High |
| Тип | Smoke, Functional |
| Предусловия | Пользователь зарегистрирован и активен |
| Тестовые данные | Валидный логин, валидный пароль |
| Статус | Not run |

| Шаг | Действие | Ожидаемый результат |
| --- | --- | --- |
| 1 | Открыть страницу авторизации | Страница открыта, форма входа отображается |
| 2 | Ввести корректный логин | Значение отображается в поле логина |
| 3 | Ввести корректный пароль | Пароль скрыт маской |
| 4 | Нажать кнопку `Войти` | Пользователь успешно авторизован |
| 5 | Проверить страницу после входа | Выполнен переход на главную страницу или в личный кабинет, отображается имя пользователя |
