Тест-кейсы  банковского перевода

## TC-MOB-BANK-001 — Перевод между своими счетами

| Поле | Значение |
|------|---------|
| Приоритет | High |
| Тип | Smoke, Functional |
| Предусловия | Пользователь авторизован, есть 2 активных счета |
| Тестовые данные | Сумма: 1000, валюта счета |
| Статус | Not run |

| Шаг | Действие | Ожидаемый результат |
|-----|----------|---------------------|
| 1 | Открыть раздел "Переводы" | Отображаются варианты переводов |
| 2 | Выбрать "Между своими счетами" | Открывается форма перевода |
| 3 | Выбрать счета списания и зачисления | Счета выбраны |
| 4 | Ввести сумму | Сумма принята |
| 5 | Подтвердить перевод | Показан экран успеха |
| 6 | Проверить балансы | Балансы обновлены корректно |
